# DCS Master v1.0.4

## 更新摘要

这个版本重点完成了两类整理：

- 重构 `SKILL.md` 主文档结构，让 skill 主文件更像正式规范而不是混合说明书
- 修正文档中的安全、信息和隐私风险点，降低公开分发时的不必要暴露

## 本次改进

- 重写 `dcs-master/SKILL.md` 主结构：
  - 明确适用范围
  - 收敛执行总则
  - 统一输出契约
  - 拆分模式选择、证据策略、图文回答、资料路由、边界约束
- 去除主文件中偏人格化、偏口语化的表达，改成更客观的技能定义
- 移除文档中的本机绝对路径引用
- 将具体联系人姓名改为角色化描述，避免公开暴露个人信息
- 收紧浏览器自动化中的敏感信息示例：
  - 不再推荐通过 `export` 长时间暴露账号密码
  - 改为当前 shell 会话临时读取
  - 明确禁止把账号、密码或安全码写入配置、脚本、缓存或版本库
- 收紧自举升级链路：
  - 不再引用 `main` 分支远程脚本
  - 改为固定提交下载
  - 增加 SHA-256 校验后再执行

## 这次解决的问题

- `SKILL.md` 结构层次不够清晰，主文件承担了过多说明性内容
- 公开仓库中包含作者本机目录路径
- 文档中包含具体姓名和支持路径，存在隐私和时效性问题
- 自举升级命令直接下载并执行 `main` 分支脚本，供应链风险偏高
- 浏览器自动化示例对敏感信息处理不够谨慎

- 图文与模式路由
- 安全与隐私边界
- 自举升级安全性
