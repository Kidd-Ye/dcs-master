# 如何修改Git密码

> 分类: 常用功能
> 来源: https://vip.kingdee.com/knowledge/536215396576301568?productLineId=40&isKnowledge=2&lang=zh-CN

---

1、首先进入项目，点击成员列表中本人所在行的“重置git密码”。

![image](https://vip.kingdee.com/download/01094c0308676a294170b7d7e3050d825e4c.png)

2、在弹出框中确认要修改的邮箱后，点击“是”。

![image](https://vip.kingdee.com/download/010921012f69aa024999abebb0c38bd7b718.png)

3、复制临时密码，点击“前往修改”。在[https://code.kingdee.com/gitlab/users/sign_in](https://code.kingdee.com/gitlab/users/sign_in) 界面填写邮箱和临时密码，点击登录。

![image](https://vip.kingdee.com/download/0109708165b70a1b4861a5904743be0a6913.png)

![image](https://vip.kingdee.com/download/010951b4ba3049fc415cb3627a09cffd20f5.png)

4、跳转到修改密码界面，在Current password处填写临时密码，在New password和Password confirmation处填写新密码后，点击Set new password按钮即可完成密码的重置。

![image](https://vip.kingdee.com/download/01090f9697703be94824bbe6c3e7ca819163.png)
