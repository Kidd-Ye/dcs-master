2 Factory Applications

* * Access Path * *

[Left navigation tree]→ [Application management]→ [Factory application library].

* * Overview * *

The developer uses the original application library interface to view and extend the purchased original applications.

* * Key features * *

-Original application viewing;

- Original application collection;

- Factory applied extensions.
![image](https://vip.kingdee.com/download/010041282c2e45f0420b9ea46e75f5dd50d3.png)
3 My Apps

* * Access Path * *

- [Left Navigation Tree]→ [Application Management]→ [My Applications];

- [View more] button in [Recently edited app] on the homepage.

* * Overview * *

The developer manages the applications under the developer through my application management interface, including new creation, export, editing, scripting, metadata, menu management, etc.

* * Key features * *

- Cloud management: new, edit, delete, svn submission, etc.;

- Application management: new, modify, delete, import, export, code management, disable/enable, script management, menu management, favorites, extensions, view;

- Application details: The application details page can be used for group management, menu management, KDE extension, application basic information, business objects, page management.
![image](https://vip.kingdee.com/download/0100ee3ecbf275b241c8a478017741cc5938.png)
4 App Actions

New Cloud

Click the left navigation to open the [My Apps] menu.Under the application list interface, the left tree represents "cloud". Click the "New" icon to create a business cloud.
![image](https://vip.kingdee.com/download/0100f9d0ac7cfc1e4b8ba65d1d7c67154932.png)
![image](https://vip.kingdee.com/download/010038028968662544f593159a190c253dd2.png)
Create a new app

Click the left tree to navigate to the cloud node where you want to create the application→ Click [New] to→ enter the application information to→ create the application successfully.
![image](https://vip.kingdee.com/download/01002ab811c9ebdf45788cb250eec0be6d87.png)
Top Actions

Actions at the top of my apps list include: New App, Extended App, Import, Export, Repository Address, svn Update/Embed, Enable, Disable, KDE Script, Menu Management, Refresh, Exit.
![image](https://vip.kingdee.com/download/0100eee78491863d492385c8b223df9588f9.png)
Menu Management

- In the Apps list, select an app and click [Menu Management] at the top to manage the menu and set the page as a menu;

- Click the [Apply Menu] button on the top toolbar to manage the menu and set the page as a menu.
![image](https://vip.kingdee.com/download/0100106ba5d17ec74e0480fb5b7f96508a9d.png)
![image](https://vip.kingdee.com/download/0100352e76b9baac4eea946763ca1b19547f.png)
App Favorites

Under the table/list view of the app list, click the [Favorites] button to favorite the app.Under the [Favorites] menu, you can quickly see your favorite apps, entities, and pages.
![image](https://vip.kingdee.com/download/01007e251d007e914344a2167cb905e1dd69.png)
![image](https://vip.kingdee.com/download/010022cfb014675a47efa248600a9f37e8f2.png)
![image](https://vip.kingdee.com/download/010069588a7d5ca44882aae54aa4558df689.png)
App Extensions & Inheritance

- Under the application list interface, click the [Extension] button on the application to extend the application;

- Under the app details screen, click the [Expand]/[Inherit] button to expand/inherit the app.
