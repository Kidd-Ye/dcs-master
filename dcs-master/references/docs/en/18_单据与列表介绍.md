Step 2: Modify the page

Open the app home page, select the function group, filter by device, filter by page type, find the page that needs to be modified, hover the mouse over the page card, click the modify icon or page encoding hyperlink, and automatically enter the designer to modify the page content.
![image](https://vip.kingdee.com/download/010060a0005abb9f47d9b56a5494d89281a1.jpg)
Step 3: Modify the document properties

- When you need to modify the field and enrich the document function, the document supports adding fields: drag and drop the field from the left control bar to the panel and configure its properties in the right property bar;

- When the document page properties need to be modified, the document also supports modifying the page properties through the selected root node. For specific property descriptions, please refer to: [Document Properties] (https://vip.kingdee.com/article/272105460826753024).As shown in the figure.
![image](https://vip.kingdee.com/download/0100123cb4e9f96a4b98b103abd6ac18f513.jpg)
Step 4: Add field and property configurations

Firmament Designer supports drag-and-drop to add fields directly from the control bar to the designer page canvas for WYSIWYG results.By checking the fields that need to be configured, you can configure the various properties of the fields on the right side of the designer.For an introduction to fields and their properties, go to the description of each field, such as: [General Fields - Introduction to Text] (https://vip.kingdee.com/article/272108654487259136).

Step 5: List Development

The document comes with a list of documents, and the pipeline data of the document form part field is displayed in the form of a list.

After creating a new document, click "List" to switch the design view to the document list designer interface.

List configuration mainly includes two aspects of operation:

- Select the entity from the entity column on the left, add it to the list of document templates in the panel, and map the form fields to the list.

- Select the entity or document template root node to seed the control's properties in the right-hand sidebar.For more information on document list properties, see [List Properties] (https://vip.kingdee.com/article/272105462470920192).
![image](https://vip.kingdee.com/download/010034d228ea5d8f4ceea1cc3a37d4a0b58b.jpg)
Step 6: Preview & Develop

- After the document design is completed, as with the dynamic form, click Save and Preview in the upper right corner to see the design effect;

- When publishing a form, you can follow these steps:

- Click the menu management icon on the application card to enter the menu management interface;

- Click Add menu item to add a page menu;

- Bind the page in the basic information;

- Switch to advanced information and configure the opening page as a document or list, corresponding to the release of the document and the document list, respectively.
