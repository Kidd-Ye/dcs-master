# DCS Master v1.0.3

## 更新摘要

这个版本重点整理了 `dcs-master/SKILL.md` 的主入口结构，让 skill 对用户和模型都更容易理解、更容易稳定执行。

## 本次改进

- 重写 `SKILL.md` 主结构：
  - 先说明“什么时候用”
  - 再明确“先做什么”
  - 再统一“首轮回复格式”
  - 再分开说明四种模式和取材顺序
- 把原本平铺的信息改成可执行导航：
  - 教练模式
  - 诊断模式
  - 修复模式
  - 浏览器协作模式
- 强化参考资料加载规则：
  - 先按问题类型选资料
  - 不再默认把整套 references 一次读完
- 收紧边界表达：
  - 明确哪些场景要重新核验
  - 明确哪些问题不要误导到 DCS 元数据发布

## 这次解决的问题

- 首次阅读 `SKILL.md` 时，用户不容易快速抓住入口
- “模式说明、资源索引、边界规则” 混排，阅读路径不稳定
- 主文件对“首轮怎么回、遇到具体操作时怎么补图文、短问句怎么路由”表达不够直接

## 版本说明

- `skill-manifest.json` 已更新到 `1.0.3`
- `README.md` 当前版本号已同步到 `v1.0.3`

## 升级建议

如果你已经安装过旧版 `dcs-master`，建议升级到 `v1.0.3`，以获得更稳定的：

- 首轮引导
- 模式切换
- 图文回答入口
- 参考资料加载顺序
